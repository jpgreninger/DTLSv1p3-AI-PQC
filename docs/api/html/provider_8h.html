Detailed Description

DTLS v1.3 Cryptographic Provider Interface.

This file defines the abstract cryptographic provider interface for DTLS v1.3, enabling multiple cryptographic backend implementations (OpenSSL, Botan, etc.) with a unified API. Supports advanced features including quantum-resistant ML-KEM cryptography, hardware acceleration, and high-performance operations.

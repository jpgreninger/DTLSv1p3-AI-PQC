Detailed Description

Abstract Crypto Operations Interface.

This interface provides a high-level abstraction layer over cryptographic operations, decoupling the protocol logic from specific crypto provider implementations. It enables easier testing, provider switching, and modular crypto operation management.

All operations maintain RFC 9147 compliance and provide consistent error handling across different crypto backends.

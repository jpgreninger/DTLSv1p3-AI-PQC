Detailed Description

Abstract base class for cryptographic providers.

This interface defines all cryptographic operations required for DTLS v1.3. Implementations provide backend-specific crypto functionality (OpenSSL, Botan, etc.)

Note
This class is thread-safe when properly implemented by concrete providers.
See also
OpenSSLProvider, BotanProvider for concrete implementations.

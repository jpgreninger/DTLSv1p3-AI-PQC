Detailed Description

Core DTLS v1.3 Protocol Types and Constants.

Defines fundamental data types, enumerations, and constants used throughout the DTLS v1.3 implementation following RFC 9147 specifications.

Key Features:

  • Type Safety: Strong typing for protocol elements
  • RFC Compliance: All types mapped to RFC 9147 specifications
  • Performance: Optimized primitive types for high-speed operations
  • Extensibility: Designed for future protocol extensions

DTLS record content types per RFC 9147 Section 5.

Defines the various types of data that can be transmitted within DTLS records, each serving a specific protocol function.

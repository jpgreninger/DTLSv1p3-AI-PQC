Detailed Description

UDP Transport implementation for DTLS v1.3

Provides multi-threaded UDP socket handling with event-driven processing, connection multiplexing, and comprehensive error handling. Abstract transport interface for DTLS

Provides a common interface for different transport implementations to be used by the DTLS protocol layer and in testing.

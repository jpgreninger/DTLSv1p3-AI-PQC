Detailed Description

Abstract Record Layer Interface

Defines the contract for DTLS v1.3 record layer operations. This interface provides complete decoupling between the connection layer and record processing, enabling modular design and improved testability.

All methods maintain RFC 9147 compliance and provide thread-safe operations.
